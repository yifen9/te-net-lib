## Development

This repository uses `just` to drive common tasks.

- Run tests:

```bash
just test
```

- Formatting:

```bash
just fmt
```

- Benchmarks:

```bash
just bench
```

- Build local documentation (pdoc):

```bash
just docs
```

- CI entry point (used by GitHub Actions):

```bash
just ci
```
